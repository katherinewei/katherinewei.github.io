# css
#### 列举不同的清除浮动的技巧

 ```HTML
   <div class="outer">
     <div class="div1"></div>
     <div class="div2"></div>
     <div class="div3"></div>
     <div class="clearfix"></div>
   </div>
 .clearfix {
   clear: both;
 }
 /* 2.为父元素增加样式 */
 .clearfix {
   overflow: auto;
   zoom: 1; // 处理兼容性
 }
 /* 3.:after 伪元素方法 (作用于父元素) */
 .outer {
   zoom: 1;
   &:after {
     display: block;
     height: 0;
     clear: both;
     content: '.';
     visibillity: hidden;
   }
 }
 ```
